# Minikube <small>image</small>

## 功能说明

Minikube本地镜像管理

## 命令

> minikube image

``` 
管理 images

Available Commands:
  build         在 minikube 中构建一个容器镜像
  load          将镜像加载到 minikube 中
  ls            列出镜像
  pull          拉取镜像
  push          推送镜像
  rm            移除一个或多个镜像
  save          从 minikube 中保存一个镜像
  tag           为镜像打标签

Use "minikube image <command> --help" for more information about a given command.
```
